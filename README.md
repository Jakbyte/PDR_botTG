**«ПДР України | Швидкий пошук»** — це зручний та швидкий Telegram-бот, створений для водіїв, кандидатів у водії та учнів автошкіл. Бот дозволяє миттєво знаходити потрібні пункти правил дорожнього руху України прямо під час руху чи навчання, без необхідності гортати довгі вебсторінки чи шукати інформацію в Google.

## ✨ Особливості проєкту

* 🚀 **Миттєвий пошук:** Просто введіть номер пункту (наприклад, `9.2` або `15.9`), і бот одразу надішле його офіційний текст.
* 📱 **Зручний UX:** Користувачеві не потрібно блукати по меню — новий пошук можна розпочати у будь-який момент, просто ввівши цифри.
* 📅 **Актуальна база:** Тексти правил повністю відповідають чинному законодавству України.
* 🛡️ **Безпека:** Архітектура проєкту побудована з використанням змінних оточення (`.env`), що виключає витік токенів.
